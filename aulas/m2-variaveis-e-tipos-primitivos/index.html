Em JS, há apenas três tipos primitivos primoridias. O number (que engloba o float e o int do Python), string (também marcada por '' ou "") e os booleanos, True ou False.

Além deles, há outros tipos primitivos importantes: dois subtipos de number: Infinity e NaN. null e undefined. object - e um subtipo array (vetor). E também há function, é um paradigma funcional da Js.

Para saber qual é o tipo primitivo em questão, digite typeof(var variável em questão)

Diferentemente de outras linguagens, o Js aceita que as variáveis mudem de tipo. Por exemplo se var n = 18 e depois var n = 'oi', não é problema.
