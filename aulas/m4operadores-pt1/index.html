Operadores aritméticos e de atribuição

Operadores

Javascript == Python

Os operadores aritméticos do Javascript são iguais àqueles que já conheço em Python e a ordem de precedência é a mesma.

  • ()
  • **
  • *, / e %
  • +, -

Operadores de atribuição.

Ao realizar n = 3 e depois n = n + 3 ou n += 3, teremos n = 6, assim como no Python.

  • n +=
  • n -=
  • n *=
  • n **=
Não há "//" para verificar resto de divisão.
